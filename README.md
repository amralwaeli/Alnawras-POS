
  # POS System Architecture Design

  This is a code bundle for POS System Architecture Design. The original project is available at https://www.figma.com/design/GVl42TVHHclYz1ApoJkqJR/POS-System-Architecture-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to GitHub Pages

  This app is configured for static hosting:

  - Vite uses relative asset paths so it can be served from a repository subpath.
  - Routing uses hash-based URLs, which avoids GitHub Pages 404 errors on refresh or deep links.

  After deployment, URLs will look like `https://<user>.github.io/<repo>/#/dashboard`.
